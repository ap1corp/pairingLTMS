# jQuery simpleClock

A tiny and simple jQuery Plugin to create an animated clock with date and time.

## Usage

```
<script type="text/javascript" src="simpleClock.min.js"></script>
```

```javascript
$(document).ready(function() {
    $("#clock").simpleClock();
});
```

```
<div id="clock">
  <div id="time"></div>
  <div id="date">/div>
</div>
```
